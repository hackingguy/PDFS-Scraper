  # PDFS-Scraper

This is a production level pdf scraper made for a company for finding files having a particular list of keywords

Can Be Used For Bulk Websites

#### Commands
```bash
  pip3 install scrapy
  git clone https://github.com/hackingguy/PDFS-Scraper
  cd PDFS-Scraper
  scrapy crawl pdfs
```
